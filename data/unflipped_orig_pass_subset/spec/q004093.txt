Design a 4-bit Up/Down Counter with the following specifications:
- The counter has two modes: Up and Down.
- It includes a reset input that sets the counter to 0.
- It includes an enable input that allows counting only when enabled.
- The Up/Down mode is controlled by a single input: when high, the counter counts up; when low, it counts down.
- The counter must wrap around when it exceeds its maximum value (15) or falls below 0.
