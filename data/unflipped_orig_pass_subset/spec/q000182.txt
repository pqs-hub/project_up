Create a Verilog module that implements a simple Bluetooth state machine. The state machine has three states: IDLE, CONNECTED, and DISCONNECTED. The system responds to two inputs: `connect` and `disconnect`. When in IDLE state, if `connect` is asserted, the state transitions to CONNECTED. If in CONNECTED state, if `disconnect` is asserted, it transitions to DISCONNECTED, and if the system is in DISCONNECTED state, it transitions back to IDLE state upon receiving a `connect` signal. The output should indicate the current state as a binary value.
