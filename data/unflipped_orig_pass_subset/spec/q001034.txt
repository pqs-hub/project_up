Design a Verilog module that performs a simple NAT (Network Address Translation) function. The module should take a 32-bit source IP address and a 32-bit destination IP address as inputs, and output the translated source and destination IP addresses based on a predefined mapping. For this example, use the following mapping:  
- Source IP: 192.168.1.10 maps to 10.0.0.10  
- Destination IP: 172.16.0.20 maps to 10.0.0.20  
The module should output the translated IP addresses when valid input addresses are given. If the input addresses do not match the mapping, output them unchanged.
