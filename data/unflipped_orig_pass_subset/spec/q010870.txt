Design a sign extension circuit that transforms a 8-bit or 16-bit input into a 32-bit output based on a control signal. If the enable signal (en) is asserted, the module checks the specified size; if size indicates BYTE (0), it extends the 8-bit input (unext[7:0]) to 32 bits by sign-extending the most significant bit (MSB). If size indicates HALF (1), it extends the 16-bit input (unext[15:0]) to 32 bits in a similar manner. If not enabled, the output remains equal to the input, effectively passing it through unchanged.
