Design a Verilog module that implements a simple state machine for an SD card interface. The state machine should have three states: IDLE, READ, and WRITE. The module should take an input signal `start`, which initiates a read or write operation depending on the `rw` signal (0 for read, 1 for write). The state transitions should occur on the rising edge of a clock signal, and the output should indicate the current state of the machine.
