Design a 2-channel DMA (Direct Memory Access) controller in Verilog. The DMA controller should have two channels (Channel 0 and Channel 1). Each channel should have a request signal (req), an acknowledge signal (ack), and a ready signal (ready). The DMA controller should respond to a request by asserting the acknowledge signal for the respective channel when it is ready to transfer data. The ready signal should indicate whether the DMA controller is available for the current operation.
