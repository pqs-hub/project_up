Implement a 4-to-16 decoder with enable. The decoder takes a 4-bit binary input and generates a one-hot encoded output.

Module name:
    decoder_4to16

Input ports:
    enable: Enable signal for the decoder. When 1, the decoder operates; when 0, all outputs are 0.
    in: 4-bit binary input to be decoded.

Output ports:
    out: 16-bit one-hot encoded output.

Implementation:
If the enable signal is asserted (enable == 1), the output is a one-hot encoded representation of the input.
If the enable signal is not asserted (enable == 0), the output is 0.
