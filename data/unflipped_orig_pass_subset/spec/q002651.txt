Design a 16-bit shift register with parallel load capability. The shift register should have the following features:
- It should shift left on a clock edge when the `shift` control signal is high.
- It should shift right on a clock edge when the `shift_right` control signal is high.
- It should load a 16-bit value when the `load` control signal is high.
- It should have a reset signal to clear the register.
- The output of the shift register should be available on an output bus.
