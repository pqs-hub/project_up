Design a combinational logic circuit that implements a simplified Huffman encoding scheme. The circuit should take a 4-bit binary input representing a character and produce a 3-bit output representing the encoded character according to the following mapping:
- 0000 -> 000
- 0001 -> 001
- 0010 -> 010
- 0011 -> 011
- 0100 -> 100
- 0101 -> 101
- 0110 -> 110
- 0111 -> 111
Any input character not included in the above mapping should output 111 (invalid encoding).
