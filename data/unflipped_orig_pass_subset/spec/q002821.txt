Design a simple NVMe controller state machine that has three states: IDLE, READ, and WRITE. The controller should transition between these states based on input signals. When in the IDLE state, it should transition to READ if the read signal is high, and to WRITE if the write signal is high. In the READ state, it should return to IDLE after one clock cycle. In the WRITE state, it should also return to IDLE after one clock cycle.
