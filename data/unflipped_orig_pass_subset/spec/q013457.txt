Design a 4-bit Gray code counter with synchronous reset. The module should have a clock input, a synchronous reset input, and a 4-bit output. The counter should generate the Gray code sequence.

Module name:
    gray_counter
Input ports:
    clk: a 1-bit clock input
    reset: a 1-bit synchronous reset input
Output ports:
    gray: a 4-bit output representing the current count value in Gray code

Implementation:
The module uses an always block triggered by the rising edge of the clock signal. Inside the always block, there is an if-else statement to check the reset signal. If the reset signal is asserted, the counter is set to 0. Otherwise, the counter increments and converts the binary count to Gray code.
