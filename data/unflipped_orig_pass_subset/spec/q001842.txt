Create a packet filtering module that determines if a given packet should be allowed or blocked based on its source and destination IP addresses. The module should have two 32-bit wide inputs for the source and destination IP addresses and a 1-bit output that indicates whether the packet is allowed (1) or blocked (0). The filtering criteria should be that packets from the source IP "192.168.1.10" to the destination IP "10.0.0.5" should be allowed, all other packets should be blocked.
