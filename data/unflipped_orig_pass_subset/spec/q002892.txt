Design a Verilog module that implements a simple Network Address Translation (NAT) functionality for IPv4 addresses. The module should map internal IP addresses (10.0.0.0/8) to a single external IP address (192.168.1.1) and manage the state of the address translation. The module should have inputs for the internal IP and a request signal, and outputs for the external IP and a valid signal indicating if the address translation is successful.
