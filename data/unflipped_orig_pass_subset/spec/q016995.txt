You have been tasked with designing a module that takes in a 5-bit input signal, performs some operations on it, and outputs a 5-bit signal. The module should be written in Verilog and should not exceed 200 lines of code. 

The module should have the following inputs: 
- `clk`: a clock signal
- `reset`: a reset signal
- `input_signal`: a 5-bit input signal

The module should have the following outputs: 
- `output_signal`: a 5-bit output signal

The module should perform the following operations on the `input_signal`: 
- If the `input_signal` is even, add 1 to it.
- If the `input_signal` is odd, subtract 1 from it.
- If the `input_signal` is 0, output 0.
