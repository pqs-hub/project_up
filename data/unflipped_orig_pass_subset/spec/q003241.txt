Design a packet filtering module for a simple firewall. The module should take in a 16-bit source IP address, a 16-bit destination IP address, and a 1-bit signal indicating whether to allow or deny the packet. The firewall should only allow packets if the source IP is within a given range (from 192.168.0.1 to 192.168.0.255) and the destination IP is 192.168.0.10. If both conditions are met, the output should be 1 (allow); otherwise, it should be 0 (deny).
