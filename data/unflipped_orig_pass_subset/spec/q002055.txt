Create a Verilog module that implements a simple PNG compression type operation. The module should take an 8-bit pixel value as input and output a 3-bit compressed value based on a basic form of color quantization. The quantization should map the pixel values to one of eight possible colors. For example, map pixel values 0-31 to 0, 32-63 to 1, and so on, up to 255 mapping to 7.
