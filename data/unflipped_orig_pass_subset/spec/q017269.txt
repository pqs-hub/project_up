You are tasked with designing a Verilog module that takes in three input signals and produces an output signal. The three input signals are A1, A2, and B1_N, while the output signal is X. The module should be named "my_module".

The module should have the following behavior:
- When A1 and A2 are both high (1), X should be high (1).
- When A1 is low (0), X should be low (0).
- When A1 is high (1) and A2 is low (0), X should be the opposite of B1_N.

You may use the following Verilog code snippet as a template for your module:
```
module my_module (
    input A1,
    input A2,
    input B1_N,
    output reg X
);

// Your code here

endmodule
```
You may assume that the input signals will be valid and stable, and that the output signal should also be stable and valid after a short delay.
