Output ports:
    rd_data1: 32-bit read data for read port 1.
    rd_data2: 32-bit read data for read port 2.

Implementation:
The module uses a 2-dimensional array to represent the 32 registers. The always block is triggered by the clock signal. If the write enable signal is high, the write data is written to the register at the write address on the rising edge of the clock. The read data is continuously assigned based on the read addresses.

Give me the complete code.

module reg_file_32x32(
