Design a 2-to-1 multiplexer using Verilog. The module should have two input ports, each capable of accepting binary values. The output port should provide a binary value that is either a or b, depending on the values of sel_b1 and sel_b2. If both sel_b1 and sel_b2 are true, the output should be b. Otherwise, the output should be a. The module must be designed using a combinational logic circuit.
