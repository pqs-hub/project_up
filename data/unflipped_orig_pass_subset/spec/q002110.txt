Create a 16-bit shift register that supports both left and right shifts based on a control signal. The control signal should determine the direction of the shift. The shift register should also support synchronous loading of a new 16-bit value when a load enable signal is active. The module should have the following ports:
- clk: Clock signal
- rst: Reset signal (active high)
- load: Load enable signal (active high)
- shift: Shift direction control signal (1 for left shift, 0 for right shift)
- din: 16-bit input data for loading into the shift register
- dout: 16-bit output data representing the current state of the shift register
