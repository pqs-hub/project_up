Design a 4-to-16 line decoder that takes a 4-bit binary input and an enable signal. When the enable signal is LOW, the decoder outputs all zeros. When the enable signal is HIGH, the decoder activates one of the 16 output lines corresponding to the binary value of the input, producing a single HIGH output that represents the decoded value. Each output line corresponds to a unique value represented by the input, effectively enabling one-hot encoding of the 4-bit input.
