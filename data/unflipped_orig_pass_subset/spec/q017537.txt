You are tasked with designing a verilog module that takes in a 16-bit input `in`, and outputs a 16-bit output `out`. The output `out` should have the same bits as the input `in`, but with the bits in reverse order. For example, if `in` is `1010101010101010`, then `out` should be `0101010101010101`.

Design a verilog module that implements this functionality. 

Your module should be named `reverse_bits` and should have the following ports:
- `input [15:0] in`: the input signal
- `output [15:0] out`: the output signal
