Design a packet filtering module for a simple firewall. The firewall will check incoming packets based on specific criteria: source IP address, destination IP address, and protocol type. Each packet will be represented by a 32-bit source IP, a 32-bit destination IP, and a 3-bit protocol type. The module should output a 'permit' signal (1) if the packet matches the allowed criteria and a 'deny' signal (0) otherwise. The criteria for this example are as follows: allow packets from the source IP 192.168.1.1 to the destination IP 10.0.0.1 with protocol type 001 (which represents TCP).
