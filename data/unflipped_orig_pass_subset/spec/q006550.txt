Design a dual-port RAM module that allows simultaneous write and read operations. When the write clock (Wclk) is active, the module captures the input data (Din) and stores it at the address specified by the write address (Waddr). At the same time, when the negative edge of the read clock (nRclk) occurs, the module outputs the data stored at the read address (Raddr) onto the output line (Dout). This configuration enables efficient data storage and retrieval in a memory application.
