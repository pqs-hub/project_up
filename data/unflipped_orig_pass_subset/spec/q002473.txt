Create a simple combinational logic circuit that performs a basic Huffman encoding operation. The circuit should take a 4-bit binary input and output a 3-bit Huffman encoded value based on static predefined mappings:
- Input 0000 -> Output 000
- Input 0001 -> Output 001
- Input 0010 -> Output 010
- Input 0011 -> Output 011
- Input 0100 -> Output 100
- Input 0101 -> Output 101
- Input 0110 -> Output 110
- Input 0111 -> Output 111
Any input not in this range should output '111' as an error.
