Design a simple PCIe endpoint controller that handles basic address decoding and read/write requests. The controller should have a 32-bit address input and should generate a valid read/write signal based on the address range. If the address falls within a specific range (e.g., 0x0000_0000 to 0xFFFF_FFFF), it should assert a valid signal. The controller must also provide a read data output and a write data input.
