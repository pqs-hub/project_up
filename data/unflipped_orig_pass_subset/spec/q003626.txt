Create a Verilog module that implements a simple SD card command response decoder. The module should take a 4-bit command input and produce a 3-bit response output based on the command. The mapping of commands to responses is as follows:  
Command (4-bit) | Response (3-bit)  
0000 | 001  
0001 | 010  
0010 | 011  
0011 | 100  
0100 | 101  
Any other command | 000
