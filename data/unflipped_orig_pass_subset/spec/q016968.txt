You are tasked with designing a Verilog module for a 4-bit adder that takes two 4-bit inputs (A and B) and produces a 4-bit output (S). The module should also have a carry-out output (COUT) that indicates whether the addition resulted in a carry-out.

The module should have the following interface:

```
module four_bit_adder (
    input [3:0] A,
    input [3:0] B,
    output [3:0] S,
    output COUT
);
```

You should use full adders to implement the 4-bit adder.
