You are tasked with designing a module that takes in a 4-bit binary input and outputs the 2's complement of the input. The module should be named "twos_comp" and have the following interface:

- 4-bit input "a"
- 4-bit output "b"

The module should use the following equation to calculate the 2's complement:

- If the input "a" is positive (most significant bit is 0), then the output "b" should be the same as input "a".
- If the input "a" is negative (most significant bit is 1), then the output "b" should be the 2's complement of input "a".

You should assume that the input "a" is a 2's complement signed binary number, and that the output "b" should also be a 2's complement signed binary number.
