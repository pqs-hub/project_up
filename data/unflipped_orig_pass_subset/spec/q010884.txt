Design a 128-entry memory module that supports both read and write operations. The module takes a 7-bit address input to specify the location in memory, a 64-bit input for data to be written (if the write signal is asserted), and provides a 64-bit output for data read from memory (if the read signal is asserted). When the memory read signal is active, the output will reflect the data stored at the specified address; when the memory write signal is active, the input data will be stored at that address.
module top_module(address,write_data,mem_read,mem_write,read_data);
