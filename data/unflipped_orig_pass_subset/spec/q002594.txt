Design a packet filtering module that checks if a packet meets specific criteria based on its source IP address, destination IP address, and protocol type. The module should output a signal to indicate whether the packet is allowed (1) or blocked (0). The filtering criteria are as follows: 
- Allow packets from source IP '192.168.1.10' to any destination IP (any protocol).
- Block packets from '192.168.1.20' to '10.0.0.1' using protocol 'TCP'.
- Allow all other packets.
