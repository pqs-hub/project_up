Design a 16-bit ADC (Analog to Digital Converter) that takes a 16-bit input representing an analog signal and provides a 4-bit output representing the range of the input signal. The output should be determined based on the following criteria: if the input is in the range of 0 to 16384 (0 to 1.0 in normalized form), the output should be 0; if the input is in the range of 16385 to 32768 (1.0 to 2.0), the output should be 1; if the input is in the range of 32769 to 49152 (2.0 to 3.0), the output should be 2; and if the input is in the range of 49153 to 65535 (3.0 to 4.0), the output should be 3. If the input is greater than 65535, the output should be 3.
