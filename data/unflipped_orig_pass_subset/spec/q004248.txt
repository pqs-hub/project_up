Design a Verilog module that implements a simple Network Address Translation (NAT) logic. The module should translate an internal IP address to an external IP address. The module should have the following inputs: an internal IP address (32 bits), an external IP address (32 bits), and a control signal to perform the translation. The output should be the translated IP address. When the control signal is asserted, the output should be the external IP address; otherwise, it should be the internal IP address.
