Design a 3-to-8 line decoder with data forwarding capabilities. The decoder should take a 3-bit input and produce an 8-bit output, where only one output line is active based on the input. Additionally, if the decoder receives an enable signal that is high, the output should reflect the decoded value. If the enable signal is low, all outputs should be low. The design should also include data forwarding logic that allows the last valid output to be reused if the enable signal falls low.
