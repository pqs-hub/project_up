Implement a 3x3 Gaussian blur filter in Verilog. The filter will take a 3x3 pixel input matrix and output a single blurred pixel value. Use the Gaussian kernel coefficients for the 3x3 matrix. The Gaussian kernel for this size can be represented as follows:
```
1  2  1
2  4  2
1  2  1
```
The normalized coefficients sum to 16, so the output pixel value will be the sum of the products of the input pixel values and the corresponding kernel coefficients divided by 16.
