Design a simple packet filtering module that checks if a packet is allowed or blocked based on the source and destination IP addresses. The module should output a signal indicating whether the packet should be allowed (`1`) or blocked (`0`). The filtering criteria are as follows:
- A packet is allowed if the source IP address is `192.168.1.1` and the destination IP address is `10.0.0.1`.
- A packet is blocked for any other IP address combination.
